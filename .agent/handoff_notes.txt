### 🔗 REST API Endpoints

```
GET  /api/autonomous/status         - System status
GET  /api/autonomous/health         - Health check
GET  /api/autonomous/director/status - Director status
POST /api/autonomous/director/start  - Start director
POST /api/autonomous/director/stop   - Stop director
GET  /api/autonomous/bmad/stats      - BMAD statistics
POST /api/autonomous/bmad/execute    - Execute BMAD cycle
POST /api/autonomous/bmad/skills     - Register skill
GET  /api/autonomous/swarm/stats     - Swarm stats
POST /api/autonomous/swarm/agents    - Register agent
DELETE /api/autonomous/swarm/agents/:id - Unregister
POST /api/autonomous/swarm/agents/:id/heartbeat - Heartbeat
GET  /api/autonomous/swarm/agents/capability/:cap - Find agents
```
