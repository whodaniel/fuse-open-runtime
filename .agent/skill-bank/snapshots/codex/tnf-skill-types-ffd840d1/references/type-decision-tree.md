# Type Decision Tree

1. Does the skill primarily modify or create other skills/workflows?
   - Yes: `meta`
2. Does it connect multiple tools/systems or data sources?
   - Yes: `integration`
3. Is the main outcome a new artifact or scaffold?
   - Yes: `construction`
4. Is the main outcome a report, trace, or explanation?
   - Yes: `diagnostic`
5. Does it enforce rules or compliance gates?
   - Yes: `governance`
6. Otherwise, it is `operational`.
