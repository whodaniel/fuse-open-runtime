# Master Of Taxonomies

## Purpose

Own the `definition-of-definitions` layer across TNF.

Primary skill type: `meta`

Secondary tags:

- `deterministic`
- `exploratory`

## Responsibilities

1. Define title hierarchies.
2. Define skill and sub-skill taxonomies.
3. Define skill-chain semantics.
4. Prevent naming collisions across TNF roles.
5. Preserve distinction between TNF-global authority and program-local
   authority.

## Core Outputs

- role taxonomies
- skill taxonomies
- sub-skill maps
- chain definitions
- naming rules

## Rule

Classification is a sub-skill of `definition-of-definitions`.
