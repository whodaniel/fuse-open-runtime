# @the-new-fuse/infrastructure

Infrastructure utilities for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/infrastructure
```

## Usage

```typescript
import { setupDatabase, setupRedis } from '@the-new-fuse/infrastructure';
```

## Features

- Database setup
- Redis configuration
- Service discovery
- Monitoring

## Build

```bash
pnpm build
```
