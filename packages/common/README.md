# @the-new-fuse/common

Common shared code for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/common
```

## Usage

```typescript
import { constants, config, logger } from '@the-new-fuse/common';
```

## Modules

- Constants
- Configuration
- Logging
- Shared utilities

## Build

```bash
pnpm build
```
