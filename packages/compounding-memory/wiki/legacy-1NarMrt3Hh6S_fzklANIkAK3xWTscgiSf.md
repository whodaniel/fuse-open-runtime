## Summary

This tutorial explains how to build an AI agent capable of performing on-chain
blockchain actions, such as sending funds and deploying smart contracts. It
utilizes the OpenAI Assistant API for conversational intelligence and the Viem
library for executing transactions via TypeScript.

## Key Points

- AI agents can be programmed to interact directly with blockchains using
  'tools' or 'functions'.
- The OpenAI Assistant framework relies on four main concepts: Assistants,
  Threads, Messages, and Runs.
- Functions act as 'superpowers' that allow the AI to interpret a user's intent
  and execute custom code.
- The development process involves setting up a TypeScript environment with
  specific libraries like Viem and dotenv.
- Crucial safety measures include using testnets and protecting private keys to
  avoid loss of real funds.
