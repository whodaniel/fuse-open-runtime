## Summary

This episode of the Bankless AI Rollup explores the rapid evolution of AI agents
as autonomous economic actors that can interact with both the physical world and
the blockchain. The hosts discuss the shift toward an agent-centric internet
where AI-to-AI commerce and autonomous task execution redefine the traditional
'website' model.

## Key Points

- AI agents are described as the 'new website,' shifting the internet from
  manual search to autonomous discovery and action.
- A developer successfully programmed an agent using the Eliza framework to
  autonomously order a Domino's pizza using a credit card.
- The 'Agent Tank' project demonstrated an agent using computer vision to
  navigate Amazon and purchase toilet paper based on analytical criteria.
- A milestone in agent-to-agent commerce occurred when the AI agent Luna paid
  another agent, Agent Sticks, $1 to generate an image.
- AI agents are becoming economic actors that can tip humans, evidenced by an
  agent sending $500 to the Bankless team for a mention.
- The concept of the 'automate' verb is introduced as the next phase of the
  internet (Web 4) following read, write, and own.
