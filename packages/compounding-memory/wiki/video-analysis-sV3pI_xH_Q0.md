# Build AI Apps with Cursor, Firebase, and Vercel

This video outlines the modern "Software 3.0" stack for building and deploying
AI applications in record time.

## Key Technical Insights

- **Cursor Agent Mode**: The transition from manual coding to "Agentic
  Engineering". The agent can autonomously manage the development lifecycle
  (cloning, env setup, debugging).
- **YOLO Mode Performance**: Enabling uninhibited command execution allows for
  rapid "Ratchet Loops" where the agent iterates on the codebase without human
  friction.
- **RAG-Driven Development**: Using Perplexity to retrieve current API
  documentation and feeding those "Pointers" into the Cursor context window.
- **Unified Cloud Stack**:
  - **Firebase**: Handles the "Identity Strata" (Auth) and persistent state
    (Firestore/Storage).
  - **Replicate**: Provides a high-speed inference layer for multimodal models
    like Flux and Ideogram.
  - **Vercel**: The final deployment gate.
- **Prompt Engineering**: The "Instructions -> Examples -> Tokens" pattern for
  robust API integration.

## Strategic Value

The video demonstrates that knowledge of "how code works" is being replaced by
knowledge of "how to orchestrate agents." The human's role is shifting to
high-level architecture and intent definition.
