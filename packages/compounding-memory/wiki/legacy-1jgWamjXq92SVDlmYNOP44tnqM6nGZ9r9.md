## Summary

This tutorial demonstrates how to build a generative UI chatbot using a Python
backend with LangGraph and a Next.js frontend. It explains how to map LLM tool
calls to specific React components and stream events in real-time to provide
immediate visual feedback to the user.

## Key Points

- Generative UI allows LLMs to trigger specific UI components instead of just
  returning plain text.
- LangGraph is used to manage the state and flow of the application through
  nodes and conditional edges.
- The architecture separates the server-side logic (Python/LangChain) from the
  client-side rendering (Next.js).
- Stream events allow the UI to show loading states as soon as a tool is
  selected, before the final data is ready.
- The implementation includes tools for fetching GitHub repository data, parsing
  invoices from images, and checking weather.

## AI Concepts
