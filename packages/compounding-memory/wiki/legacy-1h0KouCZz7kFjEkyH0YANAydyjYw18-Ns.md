## Summary

This tutorial demonstrates how to integrate real-time AI painting into Krita
using the AI Diffusion plugin and Latent Consistency Models (LCM). It covers
everything from software installation and model configuration to practical
workflows for sketching, in-painting, and maintaining character consistency.

## Key Points

- Installation of Krita 5.2+ and the AI Diffusion plugin
- Configuring the Python plugin manager and AI Image Generation docker
- Downloading and using SD 1.5 and SDXL models like Realistic Vision and
  Juggernaut XL
- Using the 'Live' mode for real-time canvas-to-AI updates via LCM
- Adjusting the strength (denoising) slider to control AI influence over
  sketches
- Utilizing layers and selection tools for targeted in-painting and editing
- Introduction to advanced workflows like character consistency and automated
  randomized prompts

## AI Concepts

Latent Consistency Models (LCM), Stable Diffusion (SD 1.5 & SDXL), ControlNet,
LoRA, Image-to-Image, In-painting, Denoising Strength, Face Detailer, Character
Consistency
