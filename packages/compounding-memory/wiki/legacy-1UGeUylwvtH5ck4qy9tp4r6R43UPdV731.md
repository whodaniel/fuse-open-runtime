## Summary

This video introduces a local LLM selection system built with Ollama and
LangChain that automatically routes user queries to the most appropriate
installed model. By employing a 'selector' LLM to evaluate model descriptions,
it eliminates the manual effort of switching between specialized tools like Code
Llama or Mistral.

## Key Points

- Automating the selection of local LLMs based on task requirements to avoid
  manual switching
- Using a 'selector' LLM (Neural Chat) to evaluate query context against a
  dictionary of available models
- Implementing a modular system that supports coding, general chat, and
  uncensored tasks
- Providing both a terminal-based implementation and a Streamlit web UI
- Integrating Ollama for efficient local model management and rapid loading
