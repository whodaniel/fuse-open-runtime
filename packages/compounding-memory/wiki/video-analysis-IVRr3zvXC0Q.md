# PromptWizard: Automated Prompt Optimization for Reasoning

Microsoft's PromptWizard framework represents the shift from manual prompt
engineering to automated, algorithmic optimization of LLM inputs.

## Key Technical Insights

- **The Optimization Loop**: Replaces human trial-and-error with an iterative
  "LLM-as-a-Judge" feedback cycle.
- **Thinking Style Mutations**: Uses "Meta-Prompts" to evolve instructions
  through diverse thinking styles (e.g., "devise an experiment," "break it down
  systematically").
- **Synthetic CoT Generation**: Automatically generates complex Chain-of-Thought
  (CoT) traces for In-Context Learning, essential for maximizing the performance
  of reasoning models like o1.
- **Joint Optimization**: Simultanously optimizes the instruction, the examples,
  and the system persona.
- **Scaling Knowledge**: Demonstrates that for specific datasets (like GSM8K), a
  generated "Expert Profile" can significantly outperform a generic system
  prompt.

## Strategic Value

The framework codifies the "Ratchet Loop" protocol. By automating the evaluation
and commitment of new prompt versions based on performance metrics, it aligns
perfectly with the TNF "Forge" vision of self-synthesizing kernels.
