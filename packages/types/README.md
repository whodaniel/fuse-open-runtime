# @the-new-fuse/types

Core type definitions for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/types
```

## Usage

```typescript
import { User, Agent, Task } from '@the-new-fuse/types';
```

## Modules

- Base types
- Entity definitions
- API response types
- Event types

## Build

```bash
pnpm build
```
