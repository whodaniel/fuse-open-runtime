# @the-new-fuse/core

Core functionality for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/core
```

## Usage

```typescript
import { FuseCore } from '@the-new-fuse/core';
```

## Features

- Agent management
- Task orchestration
- Event system
- State management

## Build

```bash
pnpm build
```
