# @the-new-fuse/api-types

Shared type definitions for The New Fuse API.

## Installation

```bash
pnpm add @the-new-fuse/api-types
```

## Usage

```typescript
import { ApiResponse, User, Agent } from '@the-new-fuse/api-types';
```

## Modules

- `auth.ts` - Authentication types
- `user.ts` - User types
- `agent.ts` - Agent types
- `task.ts` - Task types
- `workflow.ts` - Workflow types
- `metrics.ts` - Metrics types
- `api-response.ts` - API response types

## Build

```bash
pnpm build
```
