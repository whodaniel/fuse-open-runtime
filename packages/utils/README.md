# @the-new-fuse/utils

Common utility functions for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/utils
```

## Usage

```typescript
import { formatDate, validateEmail, generateId } from '@the-new-fuse/utils';
```

## Available Utilities

- Date formatting
- String manipulation
- Validation helpers
- ID generation
- Object helpers

## Build

```bash
pnpm build
```
