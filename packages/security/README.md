# @the-new-fuse/security

Security utilities for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/security
```

## Usage

```typescript
import { encrypt, decrypt, hashPassword } from '@the-new-fuse/security';
```

## Features

- Encryption/decryption
- Password hashing
- JWT utilities
- Input validation

## Build

```bash
pnpm build
```
