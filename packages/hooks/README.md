# @the-new-fuse/hooks

React hooks for The New Fuse platform.

## Installation

```bash
pnpm add @the-new-fuse/hooks
```

## Usage

```typescript
import { useAuth, useAgent, useWorkflow } from '@the-new-fuse/hooks';
```

## Available Hooks

- `useAuth` - Authentication state and methods
- `useAgent` - Agent management
- `useWorkflow` - Workflow execution
- `useApi` - API communication
- `useRealtime` - Real-time updates

## Build

```bash
pnpm build
```
