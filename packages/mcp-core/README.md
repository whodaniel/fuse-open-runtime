# @the-new-fuse/mcp-core

Model Context Protocol (MCP) core implementation for The New Fuse.

## Installation

```bash
pnpm add @the-new-fuse/mcp-core
```

## Usage

```typescript
import { MCPServer } from '@the-new-fuse/mcp-core';

const server = new MCPServer({
  name: 'my-server',
  version: '1.0.0',
});
```

## Features

- MCP server implementation
- Tool definitions
- Resource management
- Prompt templates

## Build

```bash
pnpm build
```
