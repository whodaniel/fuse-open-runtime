# Security Policy

Report suspected vulnerabilities privately to `security@thenewfuse.com`.

Please include:

- Affected domain, package, or route
- Reproduction steps
- Expected and observed behavior
- Impact assessment, if known

Do not include secrets, access tokens, customer data, or private control-plane
details in public issues or pull requests.

Public security metadata is also available at:

- https://thenewfuse.com/.well-known/security.txt
