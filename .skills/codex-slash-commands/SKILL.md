# Codex CLI Slash Commands

Codex uses slash commands in its interactive terminal user interface (TUI) to manage context, history, and settings.

## Essential Commands
- `/help`: Displays help for interactive session commands.
- `/clear`: Clears the current conversation history.
- `/compact`: Condenses conversation history to save tokens while keeping key context intact.
- `/cost`: Shows current token usage and estimated cost.
- `/model [model_name]`: Switches the active LLM model within the interactive session.
- `/exit` (or `/quit`): Exits the interactive session and returns to the standard terminal.

## Advanced Commands
- `/review`: Initiates a code review mode for the current workspace.
- `/apply`: Applies the latest code changes generated by Codex to the local tree as a git patch.
