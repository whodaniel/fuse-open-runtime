# Staff Review Signals

Use these evidence sources in each review cycle:

1. master-clock sync summary,
2. don't-die remediation summary,
3. growth blocker audit findings,
4. staff role-call and scheduling report,
5. staffing director report.

Prioritize recommendations that:

1. reduce repeated critical escalations,
2. improve ownership clarity and schedule coverage,
3. increase attribution and evidence completeness,
4. remove stale blockers that suppress throughput.
