# Skill Type Matrix

| Type | Primary Output | Typical Evidence | Example Skills |
|---|---|---|---|
| operational | Actions performed | logs, artifacts, outputs | deploy, cron, audits |
| diagnostic | Findings and explanations | reports, traces, diffs | link audit, tracing |
| construction | New scaffolding or code | new files, skeletons | app scaffolds |
| governance | Compliance and policy adherence | checks, checklists | legal, security |
| integration | Interop across systems | adapters, mappings | MCP bridging |
| meta | Skill/workflow evolution | SKILL.md updates | skill-builder |

## Notes
- Multi-output skills should be split or have a clear primary type.
