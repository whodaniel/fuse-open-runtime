# Role Gap Signals

Use these as default staffing-gap indicators:

1. High-severity blocker findings that repeat across cycles.
2. Recurring schedules without clear accountable ownership.
3. Categories with active process volume but no dedicated specialist role.
4. Persistent coordination defects (handoff, attribution, escalation).
5. New operational domains introduced without assigned role + skill.

Default prioritization order:

1. command and escalation reliability,
2. schedule and staffing coordination,
3. attribution and evidence continuity,
4. optimization and growth roles.
