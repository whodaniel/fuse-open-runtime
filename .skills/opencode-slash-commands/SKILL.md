# OpenCode Slash Commands

OpenCode provides interactive slash commands to control the agent, view statistics, and manage models.

## Core Commands
- `/help`: Prints a list of available slash commands and descriptions.
- `/clear`: Wipes the session history to restart the context window.
- `/compact`: Summarizes conversation history to save tokens without losing the primary intent of the session.
- `/cost`: Prints token consumption metrics and associated API costs.
- `/model`: Switches the active model or provider configuration.
- `/agent`: Switches the active OpenCode agent configuration.
- `/exit` (or `/quit`): Terminates the OpenCode session.
