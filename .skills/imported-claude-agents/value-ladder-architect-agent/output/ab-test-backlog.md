# A/B Test Backlog

Prioritization model: expected revenue impact x traffic volume x implementation ease.

## Top 10 tests

1. Lead magnet landing headline
- Variant A: "Build a Repeatable Growth Loop in 7 Days"
- Variant B: "Steal the 7-Day Growth Playbook"
- Metric: opt-in rate
- Duration: 72h minimum

2. Lead magnet CTA button
- A: "Send Me the Playbook"
- B: "Get Free Access"
- Metric: click-to-opt-in rate
- Duration: 72h

3. Tripwire price
- A: $27
- B: $37
- Metric: revenue per visitor
- Duration: 7 days

4. Tripwire order bump copy
- A: feature-focused bullets
- B: outcome-focused bullets
- Metric: AOV
- Duration: 7 days

5. Webinar registration headline
- A: "Predictable Growth System"
- B: "Fix Funnel Leaks Fast"
- Metric: registration rate
- Duration: 5 days

6. Webinar reminder subject line
- A: urgency framing
- B: value framing
- Metric: open rate + attendance rate
- Duration: 3 webinar cycles

7. Core offer sales page hero CTA
- A: "Join the Accelerator"
- B: "Start Implementation This Week"
- Metric: checkout start rate
- Duration: 7 days

8. Core close email sequence order
- A: Proof -> Objections -> Deadline
- B: Objections -> Proof -> Deadline
- Metric: sales conversion rate
- Duration: one full close window x 3 cohorts

9. High-ticket application page CTA
- A: "Apply for Elite Advisory"
- B: "Book a Qualification Call"
- Metric: qualified application rate
- Duration: 14 days

10. Continuity offer timing
- A: Day 10 post-core purchase
- B: Day 15 post-core purchase
- Metric: continuity start rate + 30-day retention
- Duration: 30 days

## Guardrails

- One primary metric per test
- Minimum sample size target before winner selection
- Stop-loss threshold if variant underperforms >20%
- Promote winner and archive experiment notes
