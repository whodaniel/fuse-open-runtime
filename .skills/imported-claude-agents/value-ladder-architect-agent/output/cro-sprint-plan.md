# CRO Sprint Plan

Sprint window: 30 days
Primary target: increase lead-to-core conversion while maintaining CAC efficiency.

## Step 1: Investigation and Research (Days 1-7)

Quantitative checks:
- Funnel stage drop-offs: opt-in -> tripwire -> webinar reg -> core checkout -> core purchase
- Segment splits: source, device, first-touch content, email engagement
- Outlier detection: high spend/low conversion ad sets and low-show webinar cohorts

Qualitative checks:
- Session replay/heatmap on lead magnet page and core sales page
- Form analytics for application and checkout friction
- Email click-path analysis for close sequence

Outputs:
- Bottleneck ranking table
- Evidence snapshots

## Step 2: Hypothesis Generation (Days 8-10)

Hypotheses (ICE scored):
1. If we simplify the lead magnet page hero and reduce fields, opt-in rate will improve because cognitive load drops.
2. If we move social proof above the core offer CTA, checkout starts will improve because trust is established earlier.
3. If we split webinar reminders by engagement segment, attendance will improve because message relevance increases.

Prioritization:
- H1: ICE 24
- H2: ICE 22
- H3: ICE 19

## Step 3: Optimization and Experimentation (Days 11-24)

Test plan:
- Test A: Lead magnet hero + field reduction
- Test B: Sales page social proof placement
- Test C: Webinar reminder segmentation

Guardrails:
- One primary metric per test
- Minimum sample threshold before decision
- Stop-loss if variant drops >20% against control

## Step 4: Evaluation and Learning (Days 25-30)

Decision framework:
- Promote clear winner
- Archive neutral/loser tests with learnings
- Feed findings into next backlog and copy updates

Learning log format:
- Hypothesis
- Result
- Confidence
- User behavior insight
- Next action

## Success Criteria

- +15% lead magnet opt-in rate
- +10% webinar show-up rate
- +8% core checkout-to-purchase conversion
