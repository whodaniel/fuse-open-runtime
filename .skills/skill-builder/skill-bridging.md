# Bridging Skills

Bridging is a skill type that creates deterministic handoffs between skills.

## Required Sections (BridgeSpec)

- Purpose
- Inputs (files/env/services)
- Outputs (files/logs/artifacts)
- Steps (ordered skill/tool invocations)
- Guards (pre/post conditions)
- Failure modes (expected + recovery)
- Verification (commands + expected outputs)

Use `bridge-spec-template.md`.
