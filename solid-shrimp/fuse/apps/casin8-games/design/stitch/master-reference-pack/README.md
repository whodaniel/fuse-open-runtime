# Master Reference Pack

Single-folder curated best examples across all games and shared app screens for
Stitch prompt-box reference.

Categories included:

- Slots
- Poker (multiplayer cash states)
- Tournament lobby
- Cashier / token on-ramp
- Wallet / ledger
- Roulette
- Blackjack
- Trust / fairness
- Common system & UI kit screens

Files:

- `manifest.tsv` - category, screen_id, source_dir, source_basename
- `missing.tsv` - IDs not found during assembly (if any)
- `MASTER_PROMPT_ALL_GAMES.md` - full master prompt to paste in Stitch

Naming:

- `<category>__<screen-title>__<screen-id>.png`
- `<category>__<screen-title>__<screen-id>.html`

This folder is intended for manual multi-image context loading into Stitch.
