# Workflows

The New Fuse uses a set of structured workflows to guide development and agent
interactions.

## Core Workflows

- **/alignment**: Activates the Protocol Alignment Framework.
- **/brainstorm**: Structured brainstorming for projects.
- **/codebase-audit-handoff**: Codebase Hardening & Integration Audit.
- **/create**: Create new application command.
- **/debug**: Debugging command.
- **/deploy**: Deployment command for production releases.
- **/enhance**: Add or update features.
- **/frontload**: Activates Context Frontloading System.
- **/orchestrate**: Coordinate multiple agents.
- **/plan**: Create project plan.
- **/preview**: Preview server management.
- **/status**: Display agent and project status.
- **/test**: Test generation and test running.
- **/ui-ux-pro-max**: Plan and implement UI.

These workflows are defined in `.agent/workflows/` and
`.agent/workflows/antigravity/`.
