## Data Notes (extension scope)

- Google OAuth tokens and workflow state are stored only as needed for extension
  features.
- User-initiated integrations include YouTube, AI Studio, and NotebookLM related
  flows.
- Billing is handled by TNF platform services, not legacy AIVI local backend
  code.
