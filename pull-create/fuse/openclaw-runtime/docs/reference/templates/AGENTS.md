## Agent Configuration

Each agent can be configured with:

- **Model**: The AI model to use for responses
- **Tools**: Available tools and capabilities
- **Permissions**: Access levels and restrictions
- **Custom Instructions**: Agent-specific behavior guidelines

## Usage

To use an agent, specify it in your request or configure it as your default
agent.

## Agent Development

To add a new agent:

1. Create an agent configuration file
2. Define the agent's capabilities and tools
3. Register the agent in the workspace configuration
