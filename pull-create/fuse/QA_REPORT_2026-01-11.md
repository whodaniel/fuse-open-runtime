## 🚨 Critical Issues

### Issue #1: Railway API Server Crashed

**Severity:** 🔴 Critical  
**Impact:** API-dependent pages hang indefinitely  
**Affected:** `/agents`, `/workflows`, `/tasks` (any page fetching from backend)

**Error Log from Railway:**

```
Error: Cannot find module '@the-new-fuse/core/src/services/CascadeService'
Require stack:
- /app/apps/api/dist/modules/tnf-autonomous.module.js
- /app/apps/api/dist/app.module.js
- /app/apps/api/dist/main.js
```

**Root Cause:** Import path bug in `tnf-autonomous.module.ts` - importing from
source path instead of built package export.

**Fix Applied:** ✅ Committed and pushed to `fix-pr-421` branch

- Added CascadeService export to `@the-new-fuse/core/src/index.ts`
- Changed import in `apps/api/src/modules/tnf-autonomous.module.ts` from:

  ```typescript
  // OLD (broken)
  import { CascadeService } from '@the-new-fuse/core/src/services/CascadeService';

  // NEW (fixed)
  import { CascadeService } from '@the-new-fuse/core';
  ```

**Action Required:** Merge `fix-pr-421` to main and redeploy Railway
