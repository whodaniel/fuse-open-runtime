## 📦 Installation

### From VSIX Package
```bash
# Navigate to extension directory
cd /Users/danielgoldberg/Desktop/A1-Inter-LLM-Com/The-New-Fuse/src/vscode-extension-working

# Install extension
code --install-extension the-new-fuse-7.3.0.vsix
```

### Configuration Required

1. **Set API Keys** (required for AI features):
   ```json
   // VSCode Settings (settings.json)
   {
     "theNewFuse.openai.apiKey": "your-openai-key",
     "theNewFuse.anthropic.apiKey": "your-anthropic-key"
   }
   ```

2. **Configure MCP Servers** (optional):
   ```json
   {
     "theNewFuse.mcp.servers": [
       {
         "name": "filesystem",
         "transport": "stdio",
         "command": "npx",
         "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/workspace"],
         "autoConnect": true
       }
     ]
   }
   ```
