Build and operate AI agent workflows with The New Fuse.

Use the hosted SaaS at app.thenewfuse.com, or run the local workspace from the open-source repository.

Hosted SaaS

  • Sign in to the app
  • Open onboarding
  • Open API reference

Local Install

  • Clone the repository and install dependencies with pnpm install.
  • Copy .env.example to .env and set required secrets.
  • Run pnpm run dev for local development.
  • Run pnpm run release:gate before public distribution.
