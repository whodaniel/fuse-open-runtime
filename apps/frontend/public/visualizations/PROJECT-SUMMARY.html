Lessons Learned

Technical Insights

  1. Self-contained > Dependencies - Permanent artifacts beat external dependencies
  2. Multimodal is powerful - Screenshots → Code workflow is transformative
  3. Parallel > Sequential - Multiple AI sessions beat single threaded
  4. Task > Role - Task-based agents outperform role-based
  5. Phases beat context limits - External tracking enables unlimited projects

Strategic Insights

  1. Specialize, don't generalize - Master one tool (Claude Code) deeply
  2. Delegate to AI - Gemini analysis vs. manual review
  3. Document everything - Future self will thank you
  4. Build systems, not one-offs - Reusable patterns compound
  5. Think factories, not apps - Scale is multiplicative
