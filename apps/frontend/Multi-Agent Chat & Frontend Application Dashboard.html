🎯 How to Start the Server

Method 1: Using Terminal (Recommended)

  1. Open Terminal application
  2. Copy and paste this command:
cd "$(pwd)" && pnpm run dev

Method 2: Using VS Code

  1. Open VS Code
  2. Open the frontend folder
  3. Open integrated terminal (Terminal → New Terminal)
  4. Run: pnpm run dev
(Creates a clickable .command file)

🛠️ Troubleshooting

  • Port already in use: Kill existing process with npx kill-port 3000
  • Module not found: Run pnpm install first
  • Permission denied: Try sudo pnpm run dev (Use with caution)
  • Node.js not found: Install Node.js from nodejs.org
