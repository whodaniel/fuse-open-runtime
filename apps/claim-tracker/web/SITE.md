# Project Vision: ClaimTracker

> **AGENT INSTRUCTION:** Read this file before every iteration. It serves as the
> project's "Long-Term Memory."

## 1. Core Identity

- **Project Name:** ClaimTracker
- **Mission:** Empower live sale buyers and sellers with AI-driven claim
  tracking, real-time totals, and automated reconciliation.
- **Target Audience:** Enthusiasts and resellers in live commerce (crystals,
  minerals, cards, etc.).
- **Voice:** Professional, efficient, helpful, and reliable.

## 2. Visual Language

- **The "Vibe":**
  - _Primary:_ Premium & Modern
  - _Secondary:_ Crystal-inspired (clean lines, subtle gradients)
  - _Tertiary:_ Data-focused (clear typography, high contrast for readability)

## 3. Architecture & File Structure

- **Root:** `site/public/`
- **Asset Flow:** Stitch generates to `queue/` → Validate → Move to
  `site/public/`
- **Navigation Strategy:** Sidebar for Desktop, Bottom Bar for Mobile.
