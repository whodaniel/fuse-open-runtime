# Purchase History: My Mineral Collection

A detailed list of all items claimed and confirmed across different sessions.

**DESIGN SYSTEM (REQUIRED):**

- Platform: Web, Mobile-first
- Theme: Dark, Premium, Mineral-inspired
- Background: Dark Charcoal (#121212)
- Primary Accent: Amethyst Purple (#9D50BB)
- Text Primary: White (#FFFFFF)
- Font: Inter Sans-serif
- Layout: Structured List / Data Table

**Page Structure:**

1. **Summary Header:** Lifetime spend, total items collected, and average item
   price.
2. **Filter/Search Bar:** Search by seller name or item type.
3. **Purchases Table:**
   - Column 1: Item Photo / Thumbnail.
   - Column 2: Name & Price.
   - Column 3: Seller (e.g., Jason Fulker).
   - Column 4: Date & Timestamp.
   - Column 5: "View Receipt" button (which plays the AI-extracted video clip).
4. **Export Options:** Buttons to export the list to CSV or PDF for
   taxes/inventory.
