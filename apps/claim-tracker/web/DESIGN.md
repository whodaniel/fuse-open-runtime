# Design System: ClaimTracker

## 1. Visual Theme & Atmosphere

Clean, high-fidelity dashboard with a focus on real-time data visualization.
Uses a dark theme with vibrant accents reminiscent of raw minerals.

## 2. Color Palette & Roles

- **Background:** Dark Charcoal (#121212) - Main surface
- **Surface:** Slate Gray (#1E1E1E) - Cards and Modals
- **Primary Accent:** Amethyst Purple (#9D50BB) - Confirmation and primary
  actions
- **Secondary Accent:** Teal Quartz (#6EACDA) - Claims and data highlights
- **Text Primary:** Pure White (#FFFFFF) - High emphasis
- **Text Secondary:** Silver Mist (#B0B0B0) - Secondary information

## 3. Typography Rules

- Font: Inter or Roboto (Modern Sans-serif)
- Headers: Bold, tracking -0.02em
- Body: Regular, 16px, line-height 1.5

## 4. Component Stylings

- **Buttons:** Rounded corners (8px), gradient fills for primary actions.
- **Cards:** Glassmorphism effect (subtle blur and transparency).
- **Badges:** Pill-shaped with semi-transparent backgrounds.

## 5. Design System Notes for Stitch Generation

**DESIGN SYSTEM (REQUIRED):**

- Platform: Web, Mobile-first
- Theme: Dark, Premium, Mineral-inspired
- Background: Dark Charcoal (#121212)
- Primary Accent: Amethyst Purple (#9D50BB)
- Text Primary: White (#FFFFFF)
- Font: Inter Sans-serif
- Layout: Clean Card-based Grid
