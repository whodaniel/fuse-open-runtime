# Browser Hub Audit Report - December 17, 2024

## 🔴 Critical Issues Fixed

### 1. **Extensions Button Missing** ❌ → ✅ FIXED

- **Problem**: No visible button to access Extensions panel
- **Impact**: Users couldn't manage extensions at all
- **Fix**: Added Extensions button (puzzle piece icon) to:
  - Sidebar header
  - Toolbar (browser actions area)

### 2. **Modal Visibility Bug** ❌ → ✅ FIXED

- **Problem**: Custom Input Modal, New Prompt Modal, Run Prompt Modal were
  invisible
- **Cause**: Modals used `.overlay` class which requires `.active` class for
  visibility, but JS only set `display: flex`
- **Fix**: Updated all modal open/close functions to add/remove `.active` class

### 3. **Theia IDE Removed** ✅

- Removed from Quick Access section (not ready for ecosystem)
- VS Code Web remains as the primary web IDE option

## 🆕 New Features Added

### 4. **AI Hub Section**

New sidebar section with:

- **AI Chat** - Opens AI chat panel with provider selection (Gemini, OpenAI,
  Anthropic, Local CLI)
- **Gemini** - Direct link to gemini.google.com
- **API Keys** - Panel to manage AI provider API keys (OpenAI, Anthropic,
  Gemini, Groq)
- **CLI Agents** - Configure local CLI-based AI tools (Claude CLI, Aider, etc.)

### 5. **Terminal Button**

- Added to Dev Tools section
- Opens system terminal (Terminal.app on macOS, cmd on Windows)
- IPC handler: `terminal:open`

### 6. **TNF Chrome Extension Auto-Load**

- Extension is now copied during build to `extensions/tnf-chrome-extension`
- Auto-loads on app startup via `loadChromeExtensions()`

### 7. **GitHub Passkey/WebAuthn Support**

- Added Electron command line switches to enable WebAuthn
- Addresses "partial passkey support" error when signing into GitHub
- Features enabled: `WebAuthenticationTouchIdAuthenticator`,
  `WebAuthenticationRemoteDesktopSupport`
