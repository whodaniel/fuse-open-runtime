Install Go: go install github.com/Hyaxia/blogwatcher/cmd/blogwatcher@latest Quick start blogwatcher --help Common commands Add a blog: blogwatcher add "My Blog" https://example.com List blogs: blogwatcher blogs Scan for updates: blogwatcher scan List articles: blogwatcher articles Mark an article read: blogwatcher read 1 Mark all articles read: blogwatcher read-all Remove a blog: blogwatcher remove "My Blog" Example output $ blogwatcher blogs Tracked blogs (1): xkcd URL: https://xkcd.com Copy $ blogwatcher scan Scanning 1 blog(s)... xkcd Source: RSS | Found: 4 | New: 4 Found 4 new article(s) total!
