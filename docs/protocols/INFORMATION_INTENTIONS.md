# 🎯 INFORMATION_INTENTIONS.md - Data Processing Mandates

`[CLASS:PRIME] [STATUS:LOCKED]`

This protocol defines the distinct intentions, methods, and criteria for processing information within The New Fuse (TNF) network. It ensures that data is treated according to its source's authority and intended utility.

## 1. The Personal Intelligence Layer (Sensor: YouTube)

**Source:** User-directed playlists, viewing habits, and technical channel subscriptions.
**Primary Intention:** Personal Knowledge Enrichment & "Topic Sensing."
**Processing Methods:**
- **Summarization:** Extract core technical logic from transcripts.
- **Filtering:** AI must remove fluff, ads, and channel self-promotion. 
- **Attribution:** Strictly attribute to the human creator (YouTube Channel).
**Utility:**
- **Knowledge Base:** Enriches the `AI_Knowledge_Base.md` and `KNOWLEDGE_TREE.json`.
- **Topic Leads:** Identifies new concepts (e.g., "DeepSeek R1", "MLIR") that trigger deeper research.
- **Privacy:** This layer is primarily private/personal unless a topic is deemed "Strategic."

## 2. The Authority Intelligence Layer (Sensor: Scrapers/RSS)

**Source:** Established technical news platforms (TechCrunch, The Verge, Arxiv, Official AI Blogs, GitHub Trends).
**Primary Intention:** Strategic Market Intelligence & Marketplace News Feed.
**Processing Methods:**
- **Verification:** Cross-reference search results from multiple engines (SearXNG, Exa, Tavily).
- **Impact Analysis:** Classify news as "High" (System Breaking/Major Model) or "Medium."
- **Synthesis:** Combine multiple articles into a single "Intelligence Artifact."
**Utility:**
- **Marketplace News Feed:** The primary source for the public-facing feed.
- **Swarm Action Items:** Triggers "Assimilation Tasks" for the continuous improvement loop.

## 3. The Multi-Gate Synthesis (The Bridge)

The relationship between the layers follows a specific flow:
1. **YouTube Detection:** User watches a video on a new topic.
2. **Personal Enrichment:** Transcript is distilled and added to personal knowledge.
3. **Triggered Scouting:** If the topic is technical, the **News Scout** is dispatched to find *authority news* and *scientific papers* related to that topic.
4. **Marketplace Output:** The Marketplace Feed shows the **Authority News** with a "Discovered via Personal Sensor" link if applicable.

## 4. Quality Criteria (The Attribution Cornerstone)

- **[CLASS:PRIME]:** Verified by multiple authority sources + human peer review.
- **[CLASS:ALPHA]:** Single authority source with high reputation.
- **[CLASS:BETA]:** Personal sensor discovery (YouTube/Social) - unverified.
- **[STATUS:PURGE]:** Unattributed or low-utility data.
