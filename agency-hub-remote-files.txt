## API Endpoints

### Agent Management
- `POST /api/agents/register` - Register new agent
- `DELETE /api/agents/:id` - Deregister agent
- `GET /api/agents` - List all agents
- `GET /api/agents/:id` - Get agent details
- `PUT /api/agents/:id/status` - Update agent status

### Agency Operations
- `POST /api/agencies` - Create new agency
- `GET /api/agencies` - List agencies
- `GET /api/agencies/:id` - Get agency details
- `PUT /api/agencies/:id` - Update agency
- `DELETE /api/agencies/:id` - Delete agency

### Swarm Coordination
- `POST /api/swarms` - Create swarm
- `POST /api/swarms/:id/join` - Join swarm
- `DELETE /api/swarms/:id/leave` - Leave swarm
- `POST /api/swarms/:id/coordinate` - Coordinate task
- `GET /api/swarms/:id/status` - Get swarm status

### Messaging
- `POST /api/messages/send` - Send message
- `GET /api/messages/:id` - Get message details
- `GET /api/agents/:id/messages` - Get agent messages
- `WebSocket /ws/agents/:id` - Real-time messaging

### Health and Monitoring
- `GET /health` - Health check endpoint
- `GET /metrics` - Prometheus metrics
- `GET /api/monitoring/status` - System status
- `GET /api/monitoring/agents` - Agent health status

### Required Environment Variables
```bash
# Database

REDIS_URL=redis://localhost:6379

# Authentication
JWT_SECRET=your-jwt-secret
AUTH_EXPIRY=3600

# Server
PORT=3000
HOST=0.0.0.0
NODE_ENV=production

# External Services
MCP_SERVER_URL=http://localhost:3001
WEBSOCKET_PORT=3002

# Monitoring
METRICS_ENABLED=true
LOG_LEVEL=info
PROMETHEUS_PORT=9090
```

### Prerequisites
- Node.js 18+
- PostgreSQL 13+
- Redis 6+

### Quick Start
```bash
# Clone repository
git clone https://github.com/danielgoldberg/A1-Inter-LLM-Com.git
cd A1-Inter-LLM-Com

# Install dependencies
npm install

# Setup environment
cp .env.example .env

# Run database migrations
npm run migrate

# Start development server
npm run dev

# Run tests
npm run test
```
